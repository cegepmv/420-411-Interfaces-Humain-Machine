Faisons tout d'abord le tour de l'éditeur de Unity : Hierarchy, Project, Scene, Inspector, Game, Project Manager, Asset store...

## Sprite  
Un sprite = une image affichée à l’écran dans un jeu 2D  

**Exemples de sprites :**  
- personnage
- plateforme
- pièce (coin)
- décor
- icône de vie

## Game Object  
Un game object est un objet dans la scène (peut être un sprite, une caméra, un élément UI...)

## Prefab  
Un modèle d’objet prêt à être réutilisé plusieurs fois  

## Asset store  
C’est une boutique en ligne de ressources pour Unity

* On y trouve :
- personnages 2D / 3D
- environnements (forêt, ville…)
- sons / musiques
- ffets visuels
- scripts prêts à l’emploi

## Package Manager  
C’est un outil interne à Unity pour gérer les fonctionnalités du moteur

* On y trouve :  
- systèmes officiels Unity
- outils techniques
- modules du moteur

## Tilemaps
Un Tilemap dans Unity est un système utilisé pour créer des niveaux 2D en utilisant de petits carrés réutilisables appelés tiles. Au lieu de placer les objets un par un, tu construis ton monde comme une grille.

Chaque case de la grille = une tile

**Exemples de tiles**  
sol
murs
plateformes
eau
herbe

## Application   
Choisissez un Tileset de l'asset store (2D) et créez une scène de jeu avec la palette que vous allez générer.
